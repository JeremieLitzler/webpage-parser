A smart webpage parser, for YouTube channels for example where you may want to list all the links of the vlogs.

Shrink page first?

Step 1 - How to extract the page

Simply, do CTLR+S on the webpage in your browser.

For YouTube channel, go to https://www.youtube.com/c/NAME_OF_CHANNEL/videos?view=0&sort=da&flow=grid by replacing the NAME_OF_CHANNEL by the name you need

Step 2 - Select your webpage to parse

Step 3 - Type your CSS selectors

For a YouTube channel video list, use the following (*):

  • for the list: ytd-rich-grid-renderer ytd-rich-grid-media
  • for the other, it remains unchanged. otherwise, do a little F12 and inspect the HTML to find the CSS selectors

For a search, the selectors the following (*):

  • for the list: ytd-section-list-renderer ytd-video-renderer
  • for the other, it remains unchanged. otherwise, do a little F12 and inspect the HTML to find the CSS selectors
(*) : subject to change of YouTube codebase of course
